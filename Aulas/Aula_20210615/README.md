# Aula de Eletromagnetismo I

## 15/06/2021

Tópicos dados :

- Primeira aula, com apresentação da disciplina "Eletromagnetismo I" :
  - Google Classroom/Sala de Aula e Google Drive da disciplina, em termos de conteúdo, como acessar, etc;
  - Repositório público no GitHub de "Eletromagnetismo I", em termos de conteúdo;

  + técnicas de resolução de problemas de Eletromagnestimo.
