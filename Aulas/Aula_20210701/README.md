# Aula de Eletromagnetismo I

## 01/07/2021

Tópicos previstos :

- Regras de produtos de operadores diferenciais;
- Operadores diferenciais vetoriais de 2a ordem;
- Laplaciano em coord. curvilíneas;
- Delta de Dirac;
- Gráficos 2D e 3D de campos vetoriais usando web apps e Python;
- Tirando dúvidas sobre resolução do exercício "Equações de Maxwell na Forma Integral";
- Tirando dúvidas sobre resolução do trabalho individual "Coordenadas Curvilíneas";
